1. Neural networks¶

a. Neural networks and logistic regression¶

Consider a neural network for binary classification using the sigmoid function for each unit. If the network has no hidden layer, explain why the model is equivalent to logistic regression.

If there is no hidden layer, then the logistic function directly returns the output given a set of inputs, and the weights serve the same function as a logistic regression model's coefficients. This is indistinguishable from a basic logistic regression model, which outputs a prediction using the sigmoid function given input features $x_i$ and coefficients $\beta_i$ for each feature.

b. Gradients of two-layer neural network¶

Consider a simple two-layer network as shown in the lecture slides. Given the cost function used to train the neural networks

$$ \ell(w,\alpha,\beta) = \sum_{i=1}^{m} (y^{(i)} - \sigma(w^\top z^{(i)}))^2 $$

where $\sigma(x)=\frac{1}{1 + e^{-x}}$ is the sigmoid function, show that the gradient is given by

$$ \frac{\partial \ell(w,\alpha,\beta)}{\partial w} = -\sum_{i=1}^{m} 2\left[y^{(i)} - \sigma(u^{(i)})\right]\sigma(u^{(i)})\left[1 - \sigma(u^{(i)})\right]z^{(i)} $$

where $z_1^{(i)} = \sigma(\alpha^\top x^{(i)}), z_2^{(i)} = \sigma(\beta^\top x^{(i)})$. Also find the gradient of $\ell$ with respect to $\alpha$ and $\beta$.

i. Gradient of $\ell$ with respect to $w$¶

First, let:

$$\begin{align} w^\top z^{(i)} &= u^{(i)}, \\ \sigma(w^\top z^{(i)}) &= \sigma(u^{(i)}) = \frac{1}{1 + e^{-u^{(i)}}} = (1 + e^{-u^{(i)}})^{-1}, \\ 1 - \sigma(u^{(i)}) &= \frac{1}{1 + e^{u^{(i)}}} \end{align}$$

Then:

$$\begin{align} \frac{\partial \sigma(w^\top z^{(i)})}{\partial w} &= -(1 + e^{-w^\top z^{(i)}})^{-2} (-z^{(i)} e^{-w^\top z^{(i)}}) \\ &= \frac{z^{(i)} e^{-w^\top z^{(i)}}}{(1 + e^{-w^\top z^{(i)}})^2} \\ &= z^{(i)} \frac{1}{1 + e^{-u^{(i)}}} \cdot \frac{e^{-u^{(i)}}}{1 + e^{-u^{(i)}}} \\ &= z^{(i)} \frac{1}{1 + e^{-u^{(i)}}} \cdot \frac{1}{e^{u^{(i)}} (1 + e^{-u^{(i)}})} \\ &= z^{(i)} \frac{1}{1 + e^{-u^{(i)}}} \cdot \frac{1}{1 + e^{u^{(i)}}} \\ &= z^{(i)} \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] \end{align}$$

Finally, using the derivation above and the chain rule of derivatives (noting the $-$ sign on $\sigma(w^\top z^{(i)})$:

$$\begin{align} \frac{\partial \ell(w,\alpha,\beta)}{\partial w} &= \sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(w^\top z^{(i)})\right] \frac{\partial \left[y^{(i)}-\sigma(w^\top z^{(i)})\right]}{\partial w} \\ &= -\sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(w^\top z^{(i)})\right] z^{(i)} \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] \\ &= -\sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(w^\top z^{(i)})\right] \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] z^{(i)} \end{align}$$

ii. Gradient of $\ell$ with respect to $\alpha$ and $\beta$¶

First, let:

$$\begin{align} v^{(i)} &= \alpha^\top x^{(i)}, \\ z^{(i)}_1 &= \sigma(v^{(i)}) = \sigma(\alpha^\top x^{(i)}) = \frac{1}{1 + e^{-\alpha^\top x^{(i)}}} \end{align}$$

Then:

$$\begin{align} \frac{\partial z^{(i)}_1}{\partial \alpha} &= \frac{\partial \sigma(v^{(i)})}{\partial \alpha} = \color{blue} { x^{(i)} \sigma(v^{(i)}) \left[1 - \sigma(v^{(i)})\right] } \end{align}$$

This derivation was shown in part i; simply replace $v^{(i)} = \alpha^\top x^{(i)}$ with $u^{(i)} = w^\top z^{(i)}$.

Using a similar substitution:

$$\begin{align} \frac{\partial \ell(w,\alpha,\beta)}{\partial z^{(i)}_1} &= \sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(w^\top z^{(i)})\right] \frac{\partial \left[y^{(i)}-\sigma(w^\top z^{(i)})\right]}{\partial z^{(i)}}, \\ \frac{\partial \left[y^{(i)}-\sigma(w^\top z^{(i)})\right]}{\partial z^{(i)}} &= \color{red} { -w_1 \ \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] } \end{align}$$

Finally, using the derivations above and the chain rule of derivatives:

$$\begin{align} \frac{\partial \ell(w,\alpha,\beta)}{\partial \alpha} &= \frac{\partial \ell(w,\alpha,\beta)}{\partial z^{(i)}_1} \frac{\partial z^{(i)}_1}{\partial \alpha} \\ &= \color{red}{-}\sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(w^\top z^{(i)})\right] \color{red}{w_1 \ \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right]} \color{blue} {x^{(i)} \sigma(v^{(i)}) \left[1 - \sigma(v^{(i)})\right]} \\ &= -\sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(u^{(i)})\right] \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] w_1 \ \sigma(v^{(i)}) \left[1 - \sigma(v^{(i)})\right] x^{(i)} \\ \end{align}$$

The gradient of $\ell$ with respect to $\beta$ can be derived using the same method, replacing $v^{(i)} = \alpha^\top x^{(i)}$ with $v^{(i)} = \beta^\top x^{(i)}$ and $w_1$ with $w_2$ (the element of the final layer's weight vector that corresponds to the output of the second unit in the hidden layer):

$$\begin{align} \frac{\partial \ell(w,\alpha,\beta)}{\partial \beta} &= -\sum_{i=1}^{m} 2\left[y^{(i)}-\sigma(u^{(i)})\right] \sigma(u^{(i)}) \left[1 - \sigma(u^{(i)})\right] w_2 \ \sigma(v^{(i)}) \left[1 - \sigma(v^{(i)})\right] x^{(i)} \\ \end{align}$$

This question is to implement and compare SVM and simple neural networks for the same datasets we tried for the last homework. We suggest to use scikit-learn, which is a commonly used and powerful Python library with various machine learning tools.

Both classifiers performed equally well on this dataset, regardless of step size for the neural network. My guess is that the data is linearly separable (or very close), which is an ideal case for SVM.

My observations:

  • The decision boundary for the neural network is very non-linear.
  • The decision boundary for SVM is linear.
  • Despite the difference in decision boundary shape, the best classifiers under each approach achieve the same testing accuracy.
  • SVM performed consistently well, suggesting that the dataset is linearly separable. The neural network may have performed worse with certain step sizes for two reasons:
    • The use of randomness in optimization (stochastic gradient descent).
    • The objective function of a neural network is not convex, meaning we are not guaranteed to find a global optimum - so in certain runs of the algorithm, the local optimum may be worse than the global optimum guaranteed by SVM.
